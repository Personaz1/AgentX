Usage Instructions

Follow these steps to deploy the Windows agent:

  1. Build the agent using the form above
  2. Download the generated EXE file
  3. Transfer the EXE to the target Windows system
  4. Execute the file with Administrator privileges if possible
  5. The agent will connect back to the C2 server automatically

Note: This agent is provided for testing and educational purposes only. Only deploy on systems you own or have explicit permission to test.

Capabilities

  • Execute system commands remotely
  • Upload/download files
  • Capture screenshots
  • Automatic privilege escalation
  • Persistence across reboots
